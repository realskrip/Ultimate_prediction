Десктопное приложение "Ultimate prediction" на C# и Windows Forms, в котором я коснулся вопросов асинхронного и многопоточного программирования
при работе с приложениями с UI, где UI не должен блокироваться во время выполнения каких-либо операций, для того что-бы у пользователя
не было "bad experience". также коснулся темы хранения данных в формате JSON.
Использована библиотека Newtonsoft.Json.13.0.1
